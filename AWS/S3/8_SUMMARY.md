# AWS S3 Summary

| Topic              | Description                                                   |
| ------------------ | ------------------------------------------------------------- |
| Buckets VS Objects | Global unique name, tied to a region                          |
| S3 Security        | IAM Policy, S3 Bucket Policy, Encryption                      |
| S3 Websites        | Host Static Website on AWS S3                                 |
| S3 Versioning      | Multiple versions for files, prevent accidental deletion      |
| S3 Replication     | Same Region or Cross Region, Must enable versioning           |
| S3 Storage Class   | Standard, IA, IZ-IA, Intelligent, Glacier                     |
| Snowball Family    | Import Data onto S3 through a physical device, edge computing |
| OpsHub             | Desktop application to manage Snow Family Devices             |
| Storage Gateway    | Hybrid solution to extend on-premises storage to S3           |
